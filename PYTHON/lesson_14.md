## Создание репозитория

Можно создать репозиторий на GitHub и потом клонировать его на локальный компьютер.
Можно пойти по другому пути: создать репозиторий на локальном компьютере и потом залить его на GitHub.

Мы начнем с более простой версии, сделаем репозиторий на GitHub и потом склонируем его на локальный компьютер.

### Создание репозитория на GitHub

1. Зайдите на свою страницу на GitHub.
2. Нажмите на кнопку "New" в правом верхнем углу.
3. Введите название репозитория и нажмите на кнопку "Create repository".
