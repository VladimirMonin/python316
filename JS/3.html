Метод Описание
push() Добавляет один или несколько элементов в конец массива и возвращает новую длину массива.
pop() Удаляет последний элемент из массива и возвращает его.
shift() Удаляет первый элемент из массива и возвращает его.
unshift() Добавляет один или несколько элементов в начало массива и возвращает новую длину массива.
forEach() Выполняет заданную функцию для каждого элемента массива.
map() Создает новый массив с результатами вызова указанной функции для каждого элемента исходного массива.
filter() Создает новый массив со всеми элементами, которые проходят проверку заданной функции.
reduce() Применяет функцию к аккумулятору и каждому элементу массива (слева направо), сводя его к одному значению.
join() Объединяет все элементы массива в строку.
split() Разделяет строку на массив подстрок.
slice() Возвращает поверхностную копию части массива в виде нового массива.
splice() Изменяет содержимое массива, удаляя, заменяя или добавляя элементы.
find() Возвращает первый найденный элемент, который удовлетворяет условию проверочной функции. В противном случае возвращается undefined.
indexOf() Возвращает первый индекс элемента в массиве, равный указанному значению, или -1, если таковой не найден.
lastIndexOf() Возвращает последний индекс элемента в массиве, равного указанному значению, или -1, если таковой не найден.
some() Проверяет, соответствует ли хотя бы один элемент в массиве условию, заданному в передаваемой функции.
every() Проверяет, соответствуют ли все элементы в массиве условию, заданному в передаваемой функции.
Свойство/Метод Описание
Math.E Основание натурального логарифма (приблизительно равно 2.718).
Math.LN10 Натуральный логарифм 10.
Math.LN2 Натуральный логарифм 2.
Math.LOG10E Логарифм по основанию 10 от e.
Math.LOG2E Логарифм по основанию 2 от e.
Math.PI Значение числа π (приблизительно равно 3.14159).
Math.SQRT1_2 Квадратный корень из 1/2.
Math.SQRT2 Квадратный корень из 2.
Math.abs(x) Возвращает абсолютное значение числа.
Math.ceil(x) Округляет число до ближайшего большего целого.
Math.floor(x) Округляет число вниз до ближайшего меньшего целого.
Math.round(x) Округляет число до ближайшего целого.
Math.max(...values) Возвращает наибольшее из числовых значений.
Math.min(...values) Возвращает наименьшее из числовых значений.
Math.pow(x, y) Возводит x в степень y.
Math.sqrt(x) Возвращает квадратный корень числа.
Math.random() Возвращает псевдослучайное число между 0 (включительно) и 1 (не включая 1).
Math.sin(x), Math.cos(x), Math.tan(x) Тригонометрические функции.
Math.asin(x), Math.acos(x), Math.atan(x) Обратные тригонометрические функции.
Math.log(x) Возвращает натуральный (по основанию e) логарифм числа.
Math.exp(x) Возвращает e в степени x.
