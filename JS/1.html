Правила нейминга переменных

camelCase, часто называемый lowerCamelCase, это стиль именования, при котором первое слово начинается с маленькой буквы, а каждое последующее слово или аббревиатура начинается с заглавной буквы. Этот стиль широко применяется в программировании, особенно в языках, таких как JavaScript, для именования переменных, функций и экземпляров объектов.

Примеры использования camelCase:

  • userName
  • getUserInfo
  • calculateTotalAmount

Основные преимущества использования camelCase:

  1. Читаемость: Имена переменных и функций становятся легко читаемыми, когда отдельные слова разделены различным регистром букв, вместо того чтобы использовать символы или пробелы.
  2. Совместимость: Многие языки программирования не разрешают пробелы в именах переменных или функций, и некоторые даже имеют проблемы с некоторыми символами. Использование camelCase устраняет эти проблемы.
  3. Стандарт в многих областях: Во многих языках программирования и библиотеках camelCase является стандартным стилем именования.

Однако стоит заметить, что не все сообщества и языки программирования предпочитают camelCase. Например, в Python принято использовать snake_case для переменных и функций. Тем не менее, в контексте JavaScript и некоторых других языков camelCase остается доминирующим стилем.

Вывод данных в консоль

Консоль в JavaScript предоставляет разработчикам инструменты для диагностики и отладки кода. Среди прочих методов, console предлагает три основных метода для вывода сообщений разного уровня важности: log, warn, и error.

  1. console.log()
    • Описание: Это наиболее часто используемый метод для вывода информации в консоль. Он применяется, в основном, для отладки и мониторинга выполнения кода.
    • Пример: console.log("Это обычное сообщение");
    • Визуальное отображение: В большинстве браузеров это сообщение отображается обычным черным текстом.
  2. console.warn()
    • Описание: Используется для вывода предупреждений. Подсказывает, что что-то может быть неправильно, но это не препятствует нормальной работе приложения.
    • Пример: console.warn("Это предупреждение");
    • Визуальное отображение: В большинстве браузеров предупреждения отображаются в желтом цвете, иногда с иконкой предупреждения.
  3. console.error()
    • Описание: Используется для вывода сообщений об ошибках. Указывает на серьезные проблемы, которые могут мешать выполнению программы.
    • Пример: console.error("Это сообщение об ошибке");
    • Визуальное отображение: В большинстве браузеров ошибки отображаются красным цветом с иконкой ошибки.

Методы console в JavaScript предоставляют разработчикам инструменты для отладки и вывода сообщений разного уровня важности в консоль браузера.

Математика в JS

Математические операторы в JS:

  1. Сложение (+)
    • Описание: Используется для сложения двух чисел или для конкатенации строк.
    • let result = 5 + 10; // 15
  2. Вычитание (-)
    • Описание: Используется для вычитания одного числа из другого.
    • let result = 15 - 5; // 10
  3. Умножение (*)
    • Описание: Используется для умножения двух чисел.
    • let result = 5 * 3; // 15
  4. Деление (/)
    • Описание: Используется для деления одного числа на другое.
    • let result = 15 / 3; // 5
  5. Остаток от деления (%)
    • Описание: Возвращает остаток от деления двух чисел.
    • let result = 16 % 5; // 1
  6. Целочисленное деление (//)
    • Описание: Этот оператор не существует в JS. Но можно имитировать его поведение.
    • let result = Math.floor(17 / 5); // 3
  7. Возведение в степень (**)
    • Описание: Возводит число в указанную степень.
    • let result = 5 ** 2; // 25
  8. Инкремент (++)
    • Описание: Увеличивает число на 1.
    • let x = 5; x++; // x станет 6
  9. Декремент (--)
    • Описание: Уменьшает число на 1.
    • let x = 5; x--; // x станет 4
  10. Сокращенная запись (+=, -=, *=, /=, %=, **=)
    • Описание: Позволяет быстро применять математическую операцию к переменной.
    • let x = 5; x += 10; // x станет 15

Typeof - определение типа данных

Оператор typeof в JS:

Оператор typeof возвращает строку, указывающую тип операнда.

  1. Number
    • Описание: Если операнд является числом.
    • typeof 123; // "number"
  2. String
    • Описание: Если операнд является строкой.
    • typeof "hello"; // "string"
  3. Boolean
    • Описание: Если операнд является булевым значением (true или false).
    • typeof true; // "boolean"
  4. Object
    • Описание: Если операнд является объектом (кроме функций).
    • typeof {name: "Alice"}; // "object"
  5. Undefined
    • Описание: Если операнд не был определен.
    • typeof undeclaredVar; // "undefined"
  6. Function
    • Описание: Если операнд является функцией.
    • typeof function() {}; // "function"
  7. Symbol
    • Описание: Если операнд является символом.
    • typeof Symbol("symbol"); // "symbol"
  8. BigInt
    • Описание: Если операнд является большим целым числом.
    • typeof 123n; // "bigint"
