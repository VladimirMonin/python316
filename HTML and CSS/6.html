Формы в HTML

Для чего нам нужны формы?

  • Сбор данных: Формы собирают введенную пользователями информацию и последовательна передают эту информацию для обработки.
  • Валидация данных: Формы обеспечивают проверку и валидацию введенных данным путем использования встроенных атрибутов валидации в HTML5 или с помощью JavaScript.
  • Отправка данных: Формы отправляют собранную информацию на сервер с помощью метода HTTP. Это может быть GET или POST.

Пример чек-бокса

Чекбоксы или флажки (checkboxes) позволяют пользователю многочисленно отмечать некоторые части форм, в отличие от радио-кнопок.

Пример радио-кнопок

Радио-кнопки (radio buttons) позволяют пользователю выбрать один вариант из нескольких.

Пример выпадающего списка

Выпадающие списки (drop-down lists) позволяют пользователю выбрать один вариант из нескольких.

Пример многострочного текстового поля

Многострочные текстовые поля (text areas) позволяют пользователю вводить несколько строк текста.

Пример ползунка

Ползунки (sliders) позволяют пользователю выбрать значение из некоторого диапазона.

Пример формы для отправки файла

Формы для отправки файлов (file upload forms) позволяют пользователю выбрать файлы на своем компьютере и отправить их на сервер.

iframe тег

Тег iframe позволяет встраивать веб-страницы в другие веб-страницы.
