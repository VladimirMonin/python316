Отступы в div блоках. Padding и Margin

Div как элемент верстки

Div - это блочный элемент, который используется для группировки элементов вместе, чтобы применить к ним определенные стили или для создания сетки из нескольких элементов.

Давайте посмотрим на примеры внешних и внутренних отступов в Div блоках.

Внешние отступы

Внешние отступы используются для создания пространства между элементами.

Внешние отступы могут быть установлены с помощью свойства margin.

Внешние отступы могут быть установлены с помощью одного из следующих значений:

  • auto - браузер устанавливает отступы
  • length - устанавливает отступы в px, pt, cm, etc.
  • % - устанавливает отступы в процентах от ширины контейнера
  • inherit - наследует отступы от родительского элемента

Параметры размеров блока

Div блоки могут иметь следующие параметры для указания размеров:

  • height - высота блока
  • width - ширина блока
  • min-height - минимальная высота блока
  • min-width - минимальная ширина блока
  • max-height - максимальная высота блока

Отображение нескольких Div рядом. Параметр Display

Отображение нескольких Div рядом

Div блоки по умолчанию отображаются друг под другом. Чтобы отобразить несколько Div блоков рядом, нужно использовать свойство display со значением inline или inline-block.

Display - это свойство, которое определяет, как элемент будет отображаться.

Свойство display может иметь следующие значения:

  • Block - элемент отображается как блочный элемент, начинается с новой строки и занимает всю доступную ширину.
  • Inline - элемент отображается как строчный элемент, начинается с того места, где он находится в коде и занимает только необходимую ширину.
  • Inline-block - элемент отображается как строчный элемент, начинается с того места, где он находится в коде и занимает только необходимую ширину.
  • None - элемент не отображается.
  • Flex - элемент отображается как блочный элемент, начинается с новой строки и занимает всю доступную ширину.
  • Grid - элемент отображается как блочный элемент, начинается с новой строки и занимает всю доступную ширину.

overflow - свойство, которое определяет, что делать, если содержимое не помещается в блоке

Свойство overflow может иметь следующие значения:

  • visible - содержимое не обрезается, если оно не помещается в блоке. Это значение по умолчанию.
  • hidden - содержимое обрезается, если оно не помещается в блоке.
  • scroll - добавляет полосы прокрутки, если содержимое не помещается в блоке.
  • auto - добавляет полосы прокрутки, если содержимое не помещается в блоке.

Применение Border для блоков Div. Параметры Border|border-radius|Border-style

Начнем с Border

Свойство border позволяет установить границу для элемента.

Причем мы можем установить её только с одной стороны

Свойство border может иметь следующие значения:

  • border-width - устанавливает ширину границы
  • border-style - устанавливает стиль границы
  • border-color - устанавливает цвет границы

Рамка только с одной стороны может быть сделана так:

border-left: 6px solid red;

Параметр border-style определяет стиль границы

Свойство border-style может иметь следующие значения:

  • solid - обычная линия
  • dotted - пунктирная линия
  • dashed - штриховая линия
  • double - двойная линия
  • groove - внутренняя граница
  • ridge - внешняя граница
  • inset - внутренняя граница

Параметр border-radius определяет скругления углов

Свойство border-radius может иметь следующие значения:

  • border-radius: 5px; - скругление углов на 5px
  • border-radius: 50%; - скругление углов на 50%
  • border-radius: 5px 10px 15px 20px; - скругление углов на 5px, 10px, 15px, 20px
  • border-radius: 5px 10px 15px 20px / 10px 15px 20px 25px; - скругление углов на 5px, 10px, 15px, 20px по часовой стрелке, и на 10px, 15px, 20px, 25px против часовой стрелки
