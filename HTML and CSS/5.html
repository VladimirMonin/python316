Для чего указывать разные форматы аудио и видео?

Все браузеры не поддерживают все форматы. Поэтому, чтобы наш сайт работал во всех браузерах, нужно указывать несколько форматов.

Видео

Аттрибут controls добавляет панель управления видео

Аттрибут autoplay автоматически запускает видео

Аттрибут loop зацикливает видео

Аттрибут muted отключает звук

Аттрибут weight - задает ширину

Аудио

Аттрибут controls добавляет панель управления аудио

Аттрибут autoplay автоматически запускает аудио

Аттрибут loop зацикливает аудио

Аттрибут muted отключает звук

Аттрибут weight - задает ширину

Вставка видео с Ютуба

allowfullscreen - разрешает развернуть видео на весь экран

Запретить развернуть видео на весь экран - убрать аттрибут allowfullscreen полностью

Canvas

Ваш браузер не поддерживает тег canvas.

Canvas - это элемент, который позволяет рисовать на странице. На самом деле, с помощью него можно делать анимацию, игры и многое другое. Но это уже выходит за рамки данного курса.

Caption. Thead, Tbody, Tfoot

Тег caption позволяет добавить заголовок к таблице

Тег thead позволяет выделить заголовок таблицы

Тег tbody позволяет выделить тело таблицы

Тег tfoot позволяет выделить подвал таблицы

Colgroup и col

Тег colgroup позволяет задать стили для столбцов таблицы

Тег col позволяет задать стили для столбца таблицы

Аттрибут span позволяет задать количество столбцов, которые будут объединены
