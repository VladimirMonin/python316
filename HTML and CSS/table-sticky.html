Свойства CSS для фиксированной шапки и первого столбца

Прилипание шапки и первого столбца в таблице

В данной таблице используются CSS свойства для создания эффекта фиксации шапки и первого столбца при прокрутке страницы. Рассмотрим каждый элемент по отдельности:

Шапка таблицы (<thead>):

position: sticky; делает шапку таблицы фиксированной, когда пользователь прокручивает страницу вниз. Это значит, что шапка останется видимой даже при прокрутке дальше.

top: 0; фиксирует шапку наверху страницы. Значение 0 означает, что шапка будет фиксироваться в самом верху видимой области.

z-index: 1; устанавливает порядок слоя для шапки. Это важно, чтобы шапка отображалась поверх других элементов страницы.

Первый столбец таблицы (<th> и <td> в первом столбце):

position: sticky; делает первый столбец таблицы (который содержит номера строк) фиксированным.

left: 0; фиксирует первый столбец слева. Значение 0 означает, что столбец будет прилипать к левому краю таблицы.

z-index: 1; устанавливает порядок слоя для первого столбца, аналогично шапке.

Таким образом, прилипание достигается за счет комбинации этих свойств. Шапка остается наверху страницы, а первый столбец прилипает к левому краю, обеспечивая удобство для пользователя при прокрутке больших объемов данных.
