Div как элемент блочной верстки.

Div определение.

Тег Div в HTML представляет собой блочный элемент, который обычно используется длягруппировки крупных блоков контента и их дальнейшей стилизации с помощью CSS. Он не несет в себе никакого семантического значения и служит исключительно для структурной разметки.

margin и padding отступы - важнейшие свойства блочных элементов.

В CSS отступы являются важным инструментом для создания пространства вокруг элементов, что позволяет добиться желаемой компоновки и дизайна страницы.

Отступы делятся на внешние и внутренние

Внешние margin отступы

  • Эти отступы создают пространство вокруг элемента, отделяя его от соседних элементов.
  • Внешний отступ не добавляет пространства внутри самого элемента он работает между границей элемента и соседними элементами.

Свойства для установки внешних отступов:

  • margin-topустанавливает верхний внешний отступ
  • margin-rightустанавливает правый внешний отступ
  • margin-bottomустанавливает нижний внешний отступ
  • margin-leftустанавливает левый внешний отступ
  • marginустанавливает все четыре внешних отступа одновременно

Интересный эффект, связанный с внешними отступами, — это "схлопывание отступов". Когда два вертикальных внешних отступа встречаются, они схлопываются в один, равный наибольшему из этих двух отступов.

Внутренние padding отступы

Эти отступы создают пространство внутри элемента, между содержимым элемента и его границей. Внутренний отступ влияет на размер элемента, увеличивая его визуальное пространство.

Свойства для установки внутренних отступов:

  • padding-topустанавливает верхний внутренний отступ
  • padding-rightустанавливает правый внутренний отступ
  • padding-bottomустанавливает нижний внутренний отступ
  • padding-leftустанавливает левый внутренний отступ
  • paddingустанавливает все четыре внутренних отступа одновременно

    • Порядок указания отступов в формате четырёх

    Одно значение

    margin: 10px; или padding: 10px;

    • Верхний отступ = 10px
    • Правый отступ = 10px
    • Нижний отступ = 10px
    • Левый отступ = 10px

    Все четыре стороны элемента получат одинаковый отступ.

    Два значения

    margin: 10px 20px; или padding: 10px 20px;

    • Верхний отступ = 10px
    • Правый отступ = 20px
    • Нижний отступ = 10px
    • Левый отступ = 20px

    Первое значение применяется к верхнему и нижнему отступам, второе значение — к правому и левому отступам.

    Три значения

    margin: 10px 20px 30px; или padding: 10px 20px 30px;

    • Верхний отступ = 10px
    • Правый отступ = 20px
    • Нижний отступ = 30px
    • Левый отступ = 20px

    Первое значение применяется к верхнему отступу, второе значение — к правому и левому отступам, третье значение — к нижнему отступу.

    Значения для отступов

    • auto - автоматический отступ
      • Описание: Позволяет браузеру автоматически рассчитать отступы.
      • Применение: Часто используется для центрирования блоков горизонтально. При установке margin-left: auto; и margin-right: auto; для блочного элемента с фиксированной шириной, этот элемент будет автоматически центрирован внутри своего родительского контейнера.
    • length - отступ в пикселях
      • Описание: Позволяет указать конкретный размер отступа, используя единицы измерения, такие как пиксели (px), точки (pt), сантиметры (cm) и многие другие.
      • Применение: Используется для точной настройки пространства вокруг элемента, например, margin-top: 20px; задает верхний отступ в 20 пикселей.
    • % - отступ в процентах
      • Описание: Устанавливает отступ в процентах от ширины (для горизонтальных отступов) или высоты (для вертикальных отступов) родительского элемента.
      • Применение: Если у родительского элемента ширина 500px и вы установите для дочернего элемента элемента margin-left: 10%;, то левый отступ будет равен 50px.
    • inherit - наследование отступов
      • Описание: Значение свойства будет наследовано от родительского элемента.
      • Применение: Полезно, когда вы хотите, чтобы дочерний элемент имел такие же отступы, как у его родителя, без необходимости дублировать код.

Border - границы.

Border-style - стиль рамки

Свойство border-style устанавливает стиль рамки.

Свойство border-style может принимать одно из следующих значений:

  • none - рамка отсутствует
  • hidden - рамка отсутствует
  • dotted - точечная рамка
  • dashed - пунктирная рамка
  • solid - сплошная рамка
  • double - двойная рамка
  • groove - 3D-рамка, выглядит как впадина
  • ridge - 3D-рамка, выглядит как выступ
  • inset - 3D-рамка, выглядит как внутренняя часть
  • outset - 3D-рамка, выглядит как внешняя часть
  • hidden - рамка отсутствует

border-width - толщина рамки

Свойство border-width устанавливает толщину рамки.

Свойство border-width может принимать одно из следующих значений:

  • thin - тонкая рамка
  • medium - средняя рамка
  • thick - толстая рамка
  • length - рамка заданной толщины в единицах измерения
  • % - рамка заданной толщины в процентах от ширины элемента

border-color - цвет рамки

Свойство border-color устанавливает цвет рамки.

Свойство border-color может принимать одно из следующих значений:

  • color - цвет рамки
  • transparent - прозрачная рамка

border-radius - закругление углов рамки

Свойство border-radius устанавливает радиус скругления углов рамки.

Свойство border-radius может принимать одно из следующих значений:

  • length - радиус скругления углов рамки в единицах измерения
  • % - радиус скругления углов рамки в процентах от ширины элемента

Свойства flexbox

Свойство Описание и параметры
display Устанавливает тип отображения элемента. Для flexbox используются значения:
• flex: блочный контейнер flex.
• inline-flex: строчный контейнер flex.
flex-direction Определяет главную ось контейнера.
• row: горизонтально (по умолчанию)
• row-reverse: горизонтально в обратном порядке
• column: вертикально
• column-reverse: вертикально в обратном порядке
justify-content Выравнивает элементы вдоль главной оси.
• flex-start: в начале оси
• flex-end: в конце оси
• center: по центру
• space-between: равные промежутки между элементами
• space-around: равные промежутки вокруг элементов
• space-evenly: равное распределение пространства между и вокруг элементов
align-items Выравнивает элементы вдоль поперечной оси.
• stretch: растягивает элементы на всю высоту контейнера
• flex-start: выравнивает элементы к верху контейнера
• flex-end: выравнивает элементы к низу контейнера
• center: выравнивает элементы по центру
• baseline: выравнивает элементы по базовой линии текста
flex-wrap Управляет переносом элементов на новую строку.
• nowrap: все элементы на одной строке (по умолчанию)
• wrap: элементы могут переноситься на новую строку
• wrap-reverse: элементы могут переноситься на новую строку в обратном порядке
flex Сокращенное свойство для flex-grow, flex-shrink и flex-basis.
• Первое значение: flex-grow
• Второе значение: flex-shrink
• Третье значение: flex-basis
flex-grow Определяет фактор роста элемента. Значение по умолчанию — 0. Если у всех элементов это значение равно 1, все элементы будут расти равномерно.
flex-shrink Определяет фактор сжатия элемента. Значение по умолчанию — 1. Если у всех элементов это значение равно 1, все элементы будут сжиматься равномерно.
flex-basis Определяет базовый размер элемента перед тем, как применяются flex-grow или flex-shrink. Значение по умолчанию — auto.
order Устанавливает порядок элемента внутри контейнера. Элементы с меньшим значением порядка располагаются перед элементами с большим значением. Значение по умолчанию — 0.
