Популярные параметры стилей для текст

Font size

Размер шрифта в пикселях, процентах, em, rem

Font weight

Толщина шрифта условно, жирность, от 100 до 900

Text-decorator

Декорация текста. Подчеркивание, зачеркивание, линия сверху, линия снизу

Text-transform

Преобразование текста. К верхнему регистру, к нижнему регистру, первая буква слова к верхнему регистру

Text shadow

Тень текста. Параметры: смещение по горизонтали, смещение по вертикали, размытие, цвет

White space

Разрывы строк. nowrap - не переносить строки

pre - сохраняет исходное форматирование

Text align

Выравнивание текста. left, right, center, justify

justify - выравнивание по ширине с добавлением пробелов
